The Scribe's Guide

Installation

Place the downloaded PowerTrims.jar into your server's /plugins directory and restart the server. The tome of configuration will be created for you.

Commands & Permissions

  • /powertrims panel (powertrims.panel) - Reveals the URL and secret key to the web panel.
  • /powertrims reload (powertrims.reload) - Re-reads the ancient configuration scrolls.
  • /powertrims trust <player> (powertrims.trust) - Form a pact with another player.
