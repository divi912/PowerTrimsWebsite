How to Get Ultimate Abilities

Upgrade a full set of matching trimmed armor to unlock a more powerful "Ultimate Ability." Here is how it works.

Step 1: Start the Upgrade

To begin, simply type the command /upgrade in your chat. This will open the Ultimate Upgrader window.

Step 2: Place Your Armor

Inside the Ultimate Upgrader, place a full set of armor (helmet, chestplate, leggings, and boots). All four pieces must have the same armor trim pattern (e.g., all Sentry, or all Ward).

Step 3: Initiate the Ritual

Once the full, matching set of armor is in place, click the "Initiate Ritual" button. If successful, the ritual will begin, and your armor set will be upgraded to its Ultimate form.

Configuration Guide

Learn how to customize every aspect of the PowerTrims plugin using the config.yml file.

Trim Toggles

Enable or disable specific trim abilities server-wide.

trims:
sentry: true
silence: false # Disabled
vex: true

Passive Effects

You can add, change, or remove as many passive effects as you want. To give a trim multiple effects, add another line below the first effect. To give a trim no effects, leave the trim name blank.

passive_effects:
silence:
- STRENGTH:1
- SPEED:1
dune:
- HASTE:0
- FIRE_RESISTANCE:0
ward:

After editing, simply run /powertrims reload to apply your changes!

Ability Settings

Change the specific stats for each ability, like damage, radius, and cooldown. All time values are in "ticks" (20 ticks = 1 second).

# Example for Sentry trim:
sentry:
primary:
arrow_count: 3
spread: 0.15
cooldown: 90000
true_damage: 0.5

After saving changes, run /powertrims reload in the console for them to take effect. Some changes may require a server restart.
