General Info

To gain the powerful abilities of a Power Trim, a player must be wearing a full set of armor (helmet, chestplate, leggings, and boots) that all have the same armor trim pattern applied. Mismatched pieces will not grant any effects.

Controls & Keybinds

  • Primary Abilities: To activate your armor's Primary Ability, press the default Swap Item in Hand key while sneaking (Default: Shift + F).
  • Passive Abilities: These are always active as long as you are wearing the full set of trimmed armor. No keybind is needed.
  • Disrupt Abilities: These trigger automatically under specific conditions (like taking heavy damage or falling to low health). No keybind is needed.

Commands & Permissions

The base command for the plugin is /powertrims (alias: /pt).

Command Description
/pt resetcooldown Resets all of a player's trim cooldowns.
/trust <player> Prevents abilities to harm trusted players.
/untrust <player> Removes trust from trusted player so abilities work on them.
/trustlist Shows the list of all trusted players.

How to Get Ultimate Abilities

To unlock an Ultimate Ability, you must perform a ritual:

  1. Type /upgrade
  2. Place a full set of Silence-trimmed armor into the menu.
  3. Click Initiate to begin the ritual!

IMPORTANT NOTE: You must have the required materials specified in the rituals.yml file to complete the ritual.

Ritual Configuration (rituals.yml)

rituals:
  SILENCE:
    # Set to false to disable this specific upgrade ritual.
    enabled: true
    # Duration of the ritual in seconds.
    duration: 10
    # Maximum number of times this armor set can be created. Set to -1 for no limit.
    limit: 2
    # List of materials required for the ritual, in the format 'MATERIAL_NAME:AMOUNT'.
    materials:
      - 'DIAMOND_BLOCK:1'
      - 'NETHERITE_INGOT:4'
      - 'ECHO_SHARD:8'
      - 'SCULK_CATALYST:1'

Ultimate Ability: Transformation

The ultimate ability is a transformation ability and has further sub-abilities which can be activated by Shift+7, Shift+8, and Shift+9.

Configuration Guide

Learn how to customize every aspect of the PowerTrims plugin using the config.yml file.

Trim Toggles

Enable or disable specific trim abilities server-wide.

trims:
sentry: true
silence: false # Disabled
vex: true

Passive Effects

You can add, change, or remove as many passive effects as you want. To give a trim multiple effects, add another line below the first effect. To give a trim no effects, leave the trim name blank.

passive_effects:
silence:
- STRENGTH:1
- SPEED:1
dune:
- HASTE:0
- FIRE_RESISTANCE:0
ward:

After editing, simply run /powertrims reload to apply your changes!

Ability Settings

Change the specific stats for each ability, like damage, radius, and cooldown. All time values are in "ticks" (20 ticks = 1 second).

# Example for Sentry trim:
sentry:
primary:
arrow_count: 3
spread: 0.15
cooldown: 90000
true_damage: 0.5

After saving changes, run /powertrims reload in the console for them to take effect. Some changes may require a server restart.
