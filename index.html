Give Armor Trims Real Power

PowerTrims is a simple but powerful plugin that turns cosmetic armor trims into useful abilities, adding a new level of strategy to your server.

Download the Plugin

Why Use PowerTrims?

New Gameplay Abilities

Give your players over 17 new abilities to use, based on the armor trims they find and equip. There's a power for every style of play.

Fully Customizable

You have full control. Change ability cooldowns, damage, and effects in the config file, or use the easy web panel to manage everything.

Team-Friendly

The built-in /trust system lets players team up without worrying about hitting their friends with powerful area-of-effect abilities.

All Trim Abilities

Sentry

Fires a burst of arrows that deal damage which partially ignores armor.

Silence

Creates an area where enemies cannot use Ender Pearls or their own trim abilities.

Vex

Unleashes a sonic boom that damages and weakens nearby enemies.

Ward

Grants the user temporary Absorption and Resistance effects for defense.

Wild

Launches a grapple hook to move quickly or pull targets.

Coast

Pulls in nearby enemies with a watery grasp, slowing them down.

Dune

Summons a blinding sandstorm that damages enemies over time.

Eye

Grants you "True Sight", allowing you to see invisible players and mobs through walls.

Configuration Guide

You can configure PowerTrims in two easy ways. Choose what works best for you!

1. The `config.yml` File

For direct control, you can edit the configuration file located at /plugins/PowerTrims/config.yml.

The file is structured simply. For example, to change the Sentry trim:

trim-abilities:
  sentry:
    primary:
      arrow_count: 3
      spread: 0.15
      cooldown_ticks: 90000
      true_damage: 0.5

Just change the values, save the file, and run /powertrims reload.

2. The Web Panel

For a more user-friendly experience, you can use the built-in web panel.

  1. Run /powertrims panel in-game or in the console.
  2. You will get a URL and a one-time access key.
  3. Open the URL in your browser and use the key to log in.
  4. You can now edit all settings from a clean web interface and save your changes.

Commands and Permissions

All commands start with /powertrims. Here are the main sub-commands:

  • panel - Get the link and access key for the web panel. (Permission: powertrims.panel)
  • reload - Reloads the plugin configuration. (Permission: powertrims.reload)
  • trust <player> - Allow a player to not be affected by your abilities. (Permission: powertrims.trust)
  • untrust <player> - Remove a player from your trusted list. (Permission: powertrims.trust)
  • trustlist - See all players you currently trust. (Permission: powertrims.trust)
